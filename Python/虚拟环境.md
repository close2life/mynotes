# 虚拟环境

1. 新建虚拟环境

   ```
   xdeMacBook-Pro:codes x$ python3 -m venv django
   ```


2. 激活虚拟环境

   ```
   xdeMacBook-Pro:codes x$ source django/bin/activate
   (django) xdeMacBook-Pro:codes x$
   ```

3. 安装需要的第三方模块

   ```
   pip3 install django
   ```

4. 停止虚拟环境

   ```
   (django) xdeMacBook-Pro:codes x$ deactivate
   ```
